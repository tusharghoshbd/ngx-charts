Ngx Charts Installation


Ngx Charts is an angular library for presenting data in chart. This library is easy to integrate in your angular component. This library supports Horizontal Bar, Vertical Bar, Pie, Donuts and Line chart. In this libray use D3 to calculate math, scale, axis and shape.

You can install via NPM.

npm install @tusharghoshbd/ngx-charts

Import in module file: 

        
        {{import}}
